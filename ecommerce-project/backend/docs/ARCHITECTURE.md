## 技术栈
### 后端服务
- **框架**: Spring Boot 3.2.0
- **语言**: Java 21
- **构建工具**: Maven
- **数据库**: MySQL 8.0 + JPA/Hibernate
- **API规范**: RESTful

### 容器化
- **容器引擎**: Docker
- **编排工具**: Docker Compose
- **镜像仓库**: Docker Hub / 私有仓库

### 基础设施
- **网络**: 自定义Docker网络
- **持久化**: Docker Volume
- **健康检查**: Spring Boot Actuator

## 组件说明
### 1. 后端服务 (ecommerce-backend)
- 提供商品管理的RESTful API
- 包含完整的CRUD操作
- 集成MySQL数据库
- 提供健康检查端点

### 2. 数据库服务 (MySQL)
- 使用MySQL 8.0版本
- 配置UTF-8字符集
- 数据持久化存储
- 自动初始化脚本

### 3. 容器编排
- 使用Docker Compose管理多服务
- 配置服务依赖和启动顺序
- 自定义网络隔离
- 资源限制配置

## 数据流
1. 客户端请求 → 负载均衡器
2. 负载均衡器 → 后端服务
3. 后端服务 → MySQL数据库
4. 数据库响应 → 后端服务
5. 后端响应 → 客户端
